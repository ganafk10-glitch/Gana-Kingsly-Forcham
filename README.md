# Gana Kingsly Forcham — Personal Portfolio

A fast, accessible, SEO-optimized single-page portfolio built with semantic HTML5, modern CSS3 and vanilla JavaScript.

## Folder Structure

```text
text/
├── index.html
├── css/
│   └── style.css
├── js/
│   ├── script.js
│   └── xlsx.js
├── assets/
│   ├── img/
│   │   ├── profile.jpg
│   │   └── og-image.jpg
│   └── icons/
│       └── favicon.svg
├── robots.txt
├── sitemap.xml
└── README.md
```

## Customization

1. Replace `https://your-domain.com/` in `index.html`, `robots.txt` and `sitemap.xml` with your final domain.
2. Add your real LinkedIn, GitHub and Google Scholar links in `index.html` and JSON-LD.
3. Add a professional photo as `assets/img/profile.jpg` and a 1200×630 social preview image as `assets/img/og-image.jpg`.
4. Place your downloadable CV at `assets/Gana-Kingsly-Forcham-CV.pdf` or change the CV link in the hero section.
5. Replace placeholder project cards with live GitHub repositories and case studies.

## Deployment

### Netlify
Drag the `text` folder into Netlify Drop or connect your GitHub repository. Set publish directory to `/` if the files are at repository root, or `text` if you keep this folder structure.

### Vercel
Import the GitHub repository. Use a static project. Set the output directory to `text` if needed.

### GitHub Pages
Put the content of `text` in your repository root. Go to Settings → Pages → Deploy from branch → main → root.

## SEO Checklist

- Use your full name in title, H1, meta description and structured data.
- Replace all placeholder domain and profile URLs.
- Add a real profile photo and optimized Open Graph image.
- Submit `sitemap.xml` in Google Search Console.
- Create backlinks from LinkedIn, GitHub, Google Scholar, ResearchGate and job application profiles.
- Keep page speed high: compress images, avoid heavy frameworks and keep JavaScript minimal.
- Add live project pages and internal anchors for research, machine learning and web development keywords.

## Ranking Tips

- Use one consistent name format online: `Gana Kingsly Forcham`.
- Add the portfolio link to LinkedIn, GitHub, CV, email signature and academic profiles.
- Publish 2–4 project case studies targeting keywords like `SiC defect engineering`, `machine learning engineer Norway`, `materials physicist Oslo`, and `DFT semiconductor modelling`.
- Keep the portfolio updated every month so search engines see fresh content.
